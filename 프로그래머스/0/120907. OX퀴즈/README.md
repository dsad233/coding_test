# [Level 0] OX퀴즈 - 120907

> **프로그래머스 코딩테스트 연습 - 입문**

---

### 💡 문제 설명

덧셈, 뺄셈 수식들이  
`X [연산자] Y = Z` 형태로 들어있는 문자열 배열 `quiz`가 주어집니다.

수식이 **옳다면** `"O"`  
**틀리다면** `"X"` 를 담은 배열을 `return` 하도록 함수를 작성하세요.

---

### 🔒 제한사항

- 연산 기호와 숫자 사이에는 항상 하나의 공백 존재
- 음수는 `-숫자` 형태로 표현 (공백 없음)
- 1 ≤ quiz의 길이 ≤ 10
- -10,000 ≤ X, Y ≤ 10,000
- -20,000 ≤ Z ≤ 20,000
- X, Y, Z는 0을 제외하면 앞에 0이 올 수 없음
- 연산자는 `+` 또는 `-`

---

### 📥 입출력 예

| quiz                                                       | result               |
| ---------------------------------------------------------- | -------------------- |
| ["3 - 4 = -3", "5 + 6 = 11"]                               | ["X", "O"]           |
| ["19 - 6 = 13", "5 + 66 = 71", "5 - 15 = 63", "3 - 1 = 2"] | ["O", "O", "X", "O"] |

---

### 📝 입출력 예 설명

- **3 - 4 = -3** → 틀림 → `"X"`
- **5 + 6 = 11** → 맞음 → `"O"`

---

### 🧠 풀이 방식

- 문자열을 공백 `" "` 기준으로 나누어  
  `X`, `Y`, 연산자, `Z` 추출
- 연산을 수행하여 결과와 비교
- 맞으면 `"O"`, 다르면 `"X"`
